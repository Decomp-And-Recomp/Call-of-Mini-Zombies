# Call of Mini Zombies
A decompilation of Call of Mini Zombies v4.3.4, ported to Unity 6 for compability with new phones and other general improvements like UI scaling and multiplayer working.

## Controls (PC)
|Control|Key|
|-------|---|
|Movement|`WASD`/Up Down Left Right Arrows|
|Look|Mouse Movement|
|Weapon 1|`1`|
|Weapon 2|`2`|
|Weapon 3|`3`|
|Attack|LMB|
|Buy Ammo|`R`|
|Item 1|`F`|
|Item 2|`G`|
|Pause|Pause|Escape/Enter|
